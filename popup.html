About PhishGuard

PhishGuard is an phishing detection extension that uses multiple security checks to protect you from malicious websites.

Key Features

  • Real-time URL scanning and analysis
  • Domain age verification
  • SSL certificate validation
  • Phishing heuristics detection
  • Known phishing site blacklist
  • Custom blacklist and whitelist

How It Works

PhishGuard analyzes websites you visit for common phishing indicators including:

  • Recently registered domains
  • Suspicious TLDs (.tk, .gq, etc.)
  • Missing or invalid SSL certificates
  • URLs containing IP addresses
  • Hidden form fields designed to steal credentials
  • Suspicious redirect chains
